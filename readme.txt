Trigger Netlify deployments from your dashboard.

== Description ==
Easily trigger a deploy to your Netlify site from WordPress.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Add a build hook to your Netlify project.
3. Add the Netlify build hook url on the plugin's settings page.

== How to use ==

Once a build hook url has been added to the plugin's settings page, a "Deploy" button will appear allowing you to trigger a deployment with the build hook. The plugin also keeps track of how many times content has been updated since the last deployment.
